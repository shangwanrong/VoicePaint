# VoicePaint - AI 语音绘图工具

> 纯语音控制的绘图工具，无需鼠标和键盘，只需说话就能在画布上创作。

[七牛云第四批演示视频 - Bilibili](https://www.bilibili.com/video/BV1A8Jw6cEKy/?share_source=copy_web&vd_source=939cfbf114a307fa9af1561a71fd69d9)

## 功能概览

### 基础图形绘制

支持圆、矩形、三角形、椭圆、线段、星形、箭头、文字等基础图形，可指定颜色、大小、数量。

```
"画一个圆"           → 画布中央出现圆形
"画一个红色的矩形"    → 红色矩形
"画三个蓝色的五角星"  → 3个蓝色五角星
```

### 预设模板

内置16种预设图案，一句话即可绘制。

| 类别 | 图案 |
|------|------|
| 植物/风景 | 树、房子、太阳、花 |
| 表情/符号 | 笑脸、爱心 |
| 动物（10种） | 猫、狗、鱼、蝴蝶、鸟、兔子、熊、熊猫、企鹅、青蛙 |

```
"画一棵树"    → 树的图案
"画一只猫"    → 猫的图案
"画一个笑脸"  → 笑脸图案
```

### AI SVG 生成

不在预设库中的图形，由 DeepSeek 大模型实时生成 SVG 代码并渲染到画布。

```
"画一条龙"    → AI生成龙的SVG
"画一辆汽车"  → AI生成汽车的SVG
"画一个奥特曼" → AI生成奥特曼的SVG
```

理论上可以画任何图形，不受预设库限制。

### 海龟画图（自由画笔）

逐步控制画笔绘制自由线条，支持前进、后退、转向、抬笔、落笔、画弧线。

```
"开始画"      → 进入海龟模式，出现绿色箭头指示器
"向前100"     → 前进100像素并画线
"左转90"      → 原地左转90度
"抬笔"        → 移动时不画线（指示器变灰色虚线）
"落笔"        → 移动时画线（指示器变绿色实线）
"停止画"      → 退出海龟模式，路径保存
```

### 位置控制

9个预设位置 + 方向移动，支持在指定位置画图。

```
"移到中间"          → 光标移到画布中心
"移到左上"          → 光标移到左上角
"在左边画一个圆"    → 左侧出现圆形
"在中间画一只猫"    → 中间出现猫
```

### 样式控制

```
"红色" / "蓝色" / "绿色"   → 设置颜色（20+种）
"线宽5" / "粗一点" / "细一点" → 设置线宽
"填充" / "描边"              → 切换填充/描边模式
"半透明" / "透明度50%"       → 设置透明度
"虚线" / "实线" / "点线"     → 设置线条样式
```

### 操作指令

```
"撤销" / "重做"   → 撤销/恢复操作
"清空"            → 清空画布
"删除"            → 删除上一个图形
"保存"            → 下载为PNG图片
"背景白色"        → 设置背景色
```

## 技术架构

```
┌─────────────┐     ┌─────────────┐     ┌─────────────┐
│   浏览器前端  │────→│  Express后端  │────→│  外部API     │
│             │     │  (server.js) │     │             │
│ • 录音      │     │              │     │ • 百度ASR    │
│ • Canvas渲染 │     │ • /api/baidu │     │ • DeepSeek   │
│ • 指令执行   │     │   /asr       │     │              │
│ • 意图解析   │     │ • /api/deep  │     └─────────────┘
│             │     │   seek/chat  │
│             │     │ • /api/deep  │
│             │     │   seek/svg   │
└─────────────┘     └─────────────┘
```

### 意图识别：双轨机制

```
用户说话 → 百度ASR转文字 → DeepSeek大模型解析（优先）
                              ↓ 失败
                         规则引擎关键词匹配（回退）
                              ↓ 失败
                         提示"未识别的指令"
```

- **DeepSeek大模型**：理解模糊/复杂指令，如"给我来个可爱的小动物"
- **规则引擎**：关键词匹配，不依赖网络，作为兜底方案

### 前端模块

| 文件 | 职责 |
|------|------|
| `js/app.js` | 主入口，模块初始化和事件绑定 |
| `js/baidu-recognizer.js` | 百度语音识别，录音采集和发送 |
| `js/llm-parser.js` | DeepSeek大模型意图识别 |
| `js/command-parser.js` | 规则引擎意图识别（回退） |
| `js/command-executor.js` | 指令执行，所有绘图操作 |
| `js/canvas-renderer.js` | Canvas渲染、海龟画图、SVG渲染 |
| `js/shape-factory.js` | 图形绘制工厂 |
| `js/feedback-system.js` | 语音播报 + 状态栏更新 |
| `js/action-history.js` | 操作历史（撤销/重做） |
| `js/utils.js` | 工具函数（颜色映射、位置映射等） |

### 后端

| 文件 | 职责 |
|------|------|
| `server.js` | Express服务器，代理百度ASR和DeepSeek请求，保护API密钥 |

## 快速开始

### 环境要求

- Node.js v14+
- 麦克风
- 现代浏览器（Chrome / Edge）

### 安装

```bash
git clone https://github.com/shangwanrong/VoicePaint.git
cd VoicePaint
npm install
```

### 配置密钥

在项目根目录创建 `.env` 文件：

```env
BAIDU_API_KEY=你的百度API_KEY
BAIDU_SECRET_KEY=你的百度SECRET_KEY
DEEPSEEK_API_KEY=你的DeepSeek_API_KEY
DEEPSEEK_BASE_URL=https://api.deepseek.com/v1
```

密钥获取：
- 百度智能云：https://console.bce.baidu.com/ai/#/ai/speech/app/list → 创建应用
- DeepSeek：https://platform.deepseek.com/api_keys → 创建API Key

### 启动

```bash
node server.js
```

浏览器打开 http://localhost:3000

## 使用方式

1. 点击"启动语音模式"进入画布
2. 点击底部圆形按钮开始录音（按钮变红）
3. 说出指令
4. 再次点击按钮停止录音，系统自动识别并执行

## 常见问题

| 问题 | 原因 | 解决 |
|------|------|------|
| 点击录音没反应 | 浏览器未授权麦克风 | 点击地址栏左侧图标，允许麦克风权限 |
| 语音识别返回空 | 录音太短或环境太吵 | 录音至少1秒，安静环境使用 |
| 连接被拒绝 | 服务器未启动 | 运行 `node server.js` |
| "抬笔"识别不准 | 百度ASR同音词误识别 | 已添加排比/台币等同音词处理 |
| SVG生成失败 | DeepSeek返回无效SVG | 重新说一次或换个说法 |
| DeepSeek识别不准 | 网络延迟或API额度不足 | 自动回退到规则引擎 |
